# 岁己：空响撤离

一款把类《空洞骑士》的横版动作手感与“搜打撤”风险循环结合起来的浏览器游戏原型。岁己 SUI 坠入寂羽空洞，需要把装备带进远征、搜集物资、从撤离点安全返回，并逐步重启信号回到饼干岁们的直播间。

当前 `0.2` 已是可从头玩到结局的单地图 Demo，而非静态界面稿。

## 启动

需要 Node.js 20.19+ 或 22.12+。

```powershell
npm install
npm run dev
```

浏览器打开 `http://127.0.0.1:4175`。生产构建使用：

```powershell
npm run build
npm run preview
```

## 操作

| 操作 | 键位 |
| --- | --- |
| 移动 | `A` / `D` 或方向键 |
| 跳跃 | `Space` / `W` |
| 攻击 | `J` / `X` / 鼠标左键；配合 `W` / `↑` 上劈，空中配合 `S` / `↓` 下劈反弹 |
| 黑冲 | `K` / `Shift`，需要影步靴 |
| 互动 / 拾取 / 撤离 | `E` / `Enter` |
| 地图 | `M` |
| 背包 | `Tab` |
| 使用便携修补片 | `R`，恢复 1 点蓝甲 |
| 暂停 | `Esc` / `P`，轻按 `Q` 也会打开确认页 |
| 全屏 | `F` |
| 放弃远征 | 暂停页按住 `Q` 1.2 秒，按死亡结算 |

## Demo 内容

- 简化基地：默认显示五个装备槽、4×5 随身背包和 9×10 安全仓库；工作台、拾荒交易台、线索簿与入口选择各自独立。
- 物品具有真实二维占格，例如修补片 1×2、回声核心 3×3；支持精确落点预览、旋转、自动找空位、堆叠合并、拆分、双击转移和自动整理。
- 一张 4,200×2,200 的二维房间地图，包含横向房间、回声竖井、折返支路、隐藏跳跳乐补给区和四个难度递增区域。
- 普通入口会在同一区域的多个安全点间轮换，维护电梯则直接抵达真正的深层区域。
- 跑跳、横劈、上劈、可反弹下劈、击退、受击无敌帧、武器差异、黑冲、地刺与坑洞、三类敌人与 Boss。
- 搜刮箱、材料、消耗品、装备、收藏品、地图羽片和回声核心。
- 近端与深层撤离点；安全撤离后战利品仍留在随身背包，回基地后再卸入安全仓库。
- 下一轮可在死亡地点找回一次遗失回声；再次死亡会用新回声覆盖旧回声。
- 基地不再下发任务清单；经过区域、聆听回声或发现关键物会写入线索簿，验证后自动归档，地图与新入口逐步显形。
- 羽币单件/批量买卖、整组出售、阶段性货品解锁和物品图鉴构成轻量成长/收集循环。
- 维护电梯永久捷径、深层出生点、浏览器自动存档、JSON 导入导出。
- 带回核心后需亲自前往天线墓园终端发送信号，触发首个通关动画：“结局一 · 尚未归巢”。
- 网络边界和地图注册表已抽离，后续可接权威服务器并加入更多地图。

## 设计与代码入口

- `GAME_DESIGN.md`：完整玩法、叙事、经济与后续路线。
- `DEMO_PLAN.md`：第一版范围、验收场景与技术选择。
- `src/game/RaidScene.ts`：远征世界、动作、战斗、交互与结算。
- `src/components/BaseScreen.tsx`：基地、仓库、配装和目标界面。
- `src/services/saveRepository.ts`：自动存档、导入导出和异常退出结算。
- `src/services/gameNetworkBoundary.ts`：未来联网服务的稳定接口边界。
- `tests/`：官方 Playwright 游戏客户端与完整流程专项脚本。
